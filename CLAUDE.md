# Ensemble — Claude Code Project Guide

**Ensemble** is an AI stylist app: photograph your clothes → AI tags each piece → give it a vibe ("streetwear today") → it builds an outfit from clothes you own, explains why it works, and re-picks when you push back. It uses only clothes you own.

**The canonical agent guide is [AGENTS.md](AGENTS.md) — read it before doing any work.** It defines the mandatory TDD workflow, coding standards, the AI-agent guardrails, and the documentation map. It is imported below so it is always in context.
